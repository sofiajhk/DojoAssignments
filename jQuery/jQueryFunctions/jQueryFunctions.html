.click

Click the button to set an alert!

.hide

Click the button to hide content.

.show

Click the button to show content.

.toggle

Click the button to toggle between hiding/showing the content.

.slideDown

Click the button to slide down hidden content.

.slideUp

Click the button to slide up content.

.slideToggle

Click the button to toggle between sliding up/down the content.

.fadeIn

Click button to fade-in content.

.fadeOut

Click the button to fade-out content.

.addClass

Click the button to add a class to the div (add blue background to this div)

.before

Click button to add content before an existing content.

.after

Click the button to add content after an existing content.

.append

Click the button to add content at the end of existing content.

.html

Click the button to modify existing content.

.attr

Click the button to set the value of an attribute (an image in this ex) for the 1st element in the set of matched elements.

.val

Click the button to change the value of existing content.

.text

Click the button change existing text (change p tag to h1 tag).

.data

Click the button to change content to value of the function key/attribute.
